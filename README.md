# Claude Code Shortcut 使用指南

## 简介

Claude Code Shortcut 是一款 Windows 桌面快捷工具，帮助您快速管理和发送常用命令到终端窗口。

## 功能特性

- **悬浮置顶窗口**：随时可用，不遮挡工作
- **全局热键唤起**：`Ctrl+Shift+Space` 快速呼出/隐藏
- **智能搜索**：支持命令和中文描述搜索
- **命令收藏**：常用命令置顶显示
- **分类管理**：支持按类别筛选命令
- **终端绑定**：一键发送命令到指定 CMD/PowerShell 窗口
- **使用统计**：自动记录命令使用次数，常用命令优先显示

## 安装方法

1. 下载 `release/win-unpacked/Claude Code Shortcut.exe`
2. 运行可执行文件即可使用（无需安装）
3. 首次运行会自动创建 `data.db` 数据库文件存储数据

## 使用说明

### 基本操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+Space` | 唤起/隐藏窗口 |
| `Esc` | 隐藏窗口到托盘 |
| `Ctrl+X` | 完全退出程序 |

### 添加命令

1. 点击搜索框右侧的 **+** 按钮
2. 输入命令内容（如 `git status`）
3. 输入中文描述（如 `查看git状态`）
4. 可选：输入分类（如 `git`、`cmd`、`Claude`）
5. 点击保存

### 绑定终端

1. 点击窗口顶部的 **绑定终端** 按钮
2. 选择要绑定的 CMD/PowerShell 窗口
3. 命令列表会显示 **发送** 按钮
4. 点击发送即可将命令发送到终端
5. **自动解绑**：若绑定的终端窗口被关闭，程序会自动检测并解绑，按钮显示红色提示"终端已关闭"

### 命令管理

- **复制**：点击命令右侧的 **复制** 按钮
- **编辑**：点击 **编辑** 按钮修改命令内容
- **删除**：点击 **删除** 按钮移除命令
- **收藏**：点击 ☆ 标记为收藏，收藏命令置顶显示

### 分类筛选

- 点击顶部分类标签快速筛选
- 点击分类旁的小圆点可修改分类颜色

### 主题设置

1. 点击搜索框右侧的 **太阳图标**
2. 调整背景色、文字色、强调色
3. 调整窗口大小
4. 点击 **重置为默认主题** 恢复初始设置

## 数据存储

- Windows 版本将数据存储在程序同目录的 `data.db` 文件中
- Linux AppImage 版本将数据存储在 AppImage 文件同目录的 `data.db` 文件中
- 卸载、移动或替换程序前请备份对应目录下的 `data.db`
- 升级版本时数据会自动保留

## Linux AppImage 分发

- 如果只需要对方能运行程序，通常只需要提供 `.AppImage`
- 如果还要保留当前命令和设置，请同时提供同目录的 `data.db`
- 一般不需要附带 `app.log`、`linux-unpacked/`、`release-upload/`、源码目录或 `node_modules/`
- 对方机器首次运行前需要执行 `chmod +x <AppImage>`
- 对方系统通常需要 `libfuse2` 或 `libfuse2t64`
- 请将 AppImage 放在可写目录中运行，因为程序会在同目录读写 `data.db` 和 `app.log`
- 当前产物面向 `x86_64 Linux`，ARM 设备不能直接复用

## 系统要求

- Windows 10/11
- 无需管理员权限（终端注入功能需要目标窗口有焦点）

## 常见问题

**Q: 为什么发送命令后终端没有反应？**
A: 请确保终端窗口处于可接收输入状态，且未被其他窗口遮挡。

**Q: 如何备份我的命令？**
A: 直接复制程序目录下的 `data.db` 文件即可。

**Q: 程序无法启动？**
A: 检查是否被杀毒软件拦截，将程序添加到信任列表。

## 日志文件

- Windows 版本的运行日志保存在程序同目录的 `app.log` 文件中
- Linux AppImage 版本的运行日志保存在 AppImage 文件同目录的 `app.log` 文件中
- 遇到问题可提供对应目录下的 `app.log` 以便排查
